# Team Task Manager

A Full Stack Team Task Management Web Application built to streamline project collaboration, task assignment, workflow tracking, and team productivity through role-based access control.

---

# 🚀 Features

## 🔐 Authentication & Authorization

* Secure Signup/Login system
* JWT-based authentication
* Role-based access control
* Admin & Member workflows
* Protected routes and middleware

## 📁 Project Management

* Create and manage projects
* Assign team members to projects
* Track project progress
* Manage deadlines and priorities

## ✅ Task Management

* Create, update, and delete tasks
* Assign tasks to team members
* Task status tracking:

  * Pending
  * In Progress
  * Completed
* Priority management
* Due date tracking
* Overdue task monitoring

## 📊 Dashboard & Analytics

* Real-time project statistics
* Task completion analytics
* Team productivity overview
* Pending and overdue task insights
* User activity tracking

## 🔔 Collaboration Features

* Notifications system
* Task discussions/comments
* Team workflow visibility
* Centralized task communication

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone <your-repository-url>
cd TeamTaskManager
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Start the Server

```bash
npm start
```

or

```bash
node server.js
```

After starting the server:
